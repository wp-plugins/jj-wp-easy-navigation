Easy Navigation to next and previous posts using arrow keys or navigation buttons.

== Description ==

A simple navigation plugin which helps users to move backward and forward posts using arrow keys or buttons. It is developed after getting inspiration from makeuseof.com post pages. The plugin will show navigation buttons only on single post pages. On hovering the button, next/previous post details are shown in a neat box.

Visit [Plugin Home Page](http://www.jobyj.in/wordpress/jj-wp-easy-navigation-wordpress-plugin "jj-WP Easy Navigation") for more support.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the entire jj-WP-easy-navigation.zip directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. Previous Post button which will be placed fixed vertically centered on left side. It will be displayed only on single post pages.
2. Button on mouseover displays the post details in a popup.
